# Affirmation Studio

A polished single-page affirmations app built with plain HTML, CSS, and JavaScript.

## Features

- Curated affirmation categories
- Daily seeded affirmation per category
- Save favorites with local storage
- Recent history tracking
- Copy-to-clipboard support
- Fully responsive glassmorphism-inspired layout

## Run

Open `index.html` directly in a browser, or start the local dev server:

```bash
npm run dev
```

Then visit `http://localhost:3000`.
